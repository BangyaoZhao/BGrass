# BGrass
## Installation

```
devtools::install_github("BangyaoZhao/BGrass")
```

## Vignette

We have a vignette in this package that provides a reproducible example of using BGrass. 

```
devtools::install_github("BangyaoZhao/BGrass",
                         build_vignettes = TRUE,
                         force = TRUE)
vignette('toy_example', package = 'BGrass')
```
